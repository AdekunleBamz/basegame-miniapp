## [1.0.0] - 2025-12-07

### Added
- Initial release of BaseArcade
- Smart contract integration on Base mainnet
- Arcade runner game with keyboard controls
- Real-time leaderboard with player rankings
- Wallet connection via RainbowKit
- 7-day game cycle with automatic resets
- Prize pool winner takes all mechanism
- Farcaster Frame integration for social sharing
- Responsive design for mobile and desktop
- Game instructions modal
- Error boundary for graceful error handling
- Toast notifications for user feedback
- SEO optimization with meta tags and OG images
- PWA support with manifest
- Footer with social links
- Security policy and vulnerability reporting
- CI/CD pipeline with GitHub Actions
- Comprehensive documentation

### Features
- **Entry Fee**: 0.0001 ETH to join games
- **Game Duration**: 7 days per cycle
- **Prize Distribution**: Winner takes entire pot
- **Score Tracking**: On-chain score verification
- **Leaderboard**: Real-time rankings
- **Mobile Support**: Fully responsive design
- **Dark Mode**: Automatic theme detection

### Security
- Audited smart contract
- No admin keys or backdoors
- Transparent on-chain logic
- Secure wallet integration

### Technical
- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- Wagmi + Viem for Web3 integration
- RainbowKit for wallet connection
- Base blockchain (Chain ID: 8453)
